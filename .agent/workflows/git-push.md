---
description: Commit and push changes to GitHub
---

1. Check the current git status to review all changes:
```bash
cd /home/rickard/Development/Veriqko/Veriqko && git status
```

2. Stage all changes (or specify individual files):
```bash
cd /home/rickard/Development/Veriqko/Veriqko && git add -A
```

3. Commit with a descriptive message (replace "your message" with an appropriate commit message):
```bash
cd /home/rickard/Development/Veriqko/Veriqko && git commit -m "your message"
```

4. Push to the remote repository:
```bash
cd /home/rickard/Development/Veriqko/Veriqko && git push
```
